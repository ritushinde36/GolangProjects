JWT authentication in go
tutorial that I followed - https://www.youtube.com/watch?v=ma7rUS_vW9M&t=910s

install the dependencies

1) routing -- gin
2) DB - gorm, mysql
3) secrets - goenv
4) jwt - golang-jwt
5) enctrytion and decryption - bcrypt
six) restart project upon changes in file - compile deamon
