create a command line application which is a quiz game. 
This quiz game is a cli tool and you can run it with ./quiz or quiz
data from the quiz should come from a csv file,
there are 4 kinds of problems, addition subtraction multiplication and division
each should have their own csv files
if you want to solve addition problems, then use the -add flag
if you want to solve subtraction problems, then use the -sub flag
if you want to solve multiplication problems, then use the -mul flag
if you want to solve division problems, then use the -div flag flag

you also have to pass the name of the file in the -csv flag ex - -csv "addition_problems.csv"

it should also have a time limit -limit in seconds, once the time limit is crossed, the quix should exit out and display the final score

the score should be displayed at the end of the script
score in blue
then it should display the problem numbers that you got right and the problem numbers that you got wrong
correct answers in green
and incorrect answers in red.
